This is just a simple and better test analysis website made using google's aistudio for my school tests.

## Use it for your own test analysis
### Download the json
Download the json from the school test page from the networks tab in devtools

### Replace the json
Download this code and replace the json in `/public/`. 

### Add more tests
To add more tests, place the json and update it in `/src/constants.tsx`

### To run it
Paste these commands to run the website.
``npm install``
``npm run dev``
